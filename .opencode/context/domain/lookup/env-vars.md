# Environment Variable Catalog

Source: `deploy/.env.example`

## Database
`POSTGRES_USER`, `POSTGRES_PASSWORD`, `POSTGRES_DB`, `POSTGRES_HOST`, `POSTGRES_PORT`

## RabbitMQ
`RABBITMQ_DEFAULT_USER`, `RABBITMQ_DEFAULT_PASS`, `RABBITMQ_HOST`
`RABBITMQ_QUEUE_MATCH_IDS`, `RABBITMQ_QUEUE_DLQ`
`RABBITMQ_QUEUE_RAW_MATCHES`, `RABBITMQ_QUEUE_RAW_MATCHES_DLQ`

**Per-service RabbitMQ credentials** (least-privilege users):
`RABBITMQ_USER_ID_FETCHER`, `RABBITMQ_USER_ID_FETCHER_PASS`
`RABBITMQ_USER_DETAIL_FETCHER`, `RABBITMQ_USER_DETAIL_FETCHER_PASS`
`RABBITMQ_USER_PARSER`, `RABBITMQ_USER_PARSER_PASS`

## Redis
`REDIS_HOST`, `REDIS_PORT`, `REDIS_PASSWORD`, `REDIS_DB`

## Proxy Manager
Pool: `PROXY_POOL_MAX_SIZE` (2000), `PROXY_POOL_MIN_SIZE` (20)
Validation: `PROXY_VALIDATION_CONCURRENCY` (150), `PROXY_VALIDATION_TIMEOUT_SEC` (10)
Refresh: `PROXY_REFRESH_INTERVAL_MIN` (15), `PROXY_REFRESH_SOURCE_URL`, `PROXY_REFRESH_SOURCE_FETCH_TIMEOUT_SEC` (30)
Leasing: `PROXY_LEASE_DURATION_SEC` (120), `PROXY_COOLDOWN_MIN` (5)
Revalidation: `PROXY_REVALIDATION_INTERVAL_MIN` (60) — full pool revalidation, removes dead proxies
Observability: `PROXY_METRICS_PORT` (9090)
Lifecycle: `PROXY_SHUTDOWN_GRACE_MS` (15000)
Source: `PROXY_FILE_PATH`, `PROXY_REFRESH_SOURCE_URL`, `PROXY_SOURCE_USER_AGENT`

## ID Fetcher
`FETCH_SCHEDULE` (`*/5 * * * *` — every 5 min for continuous ingestion), `FETCH_LAST_COUNT_DAY` (30), `FETCH_LOBBY_TYPES` (1,2,6)
`ID_FETCHER_START_RUN`, `ID_FETCHER_START_RUN_MIN_POOL_SIZE` (20), `ID_FETCHER_START_RUN_MAX_WAIT` (5m)
`ID_FETCHER_METRICS_PORT` (9094), `ID_FETCHER_WATERMARK_LOOKBACK_DAYS` (defaults to `FETCH_LAST_COUNT_DAY`)

## Detail Fetcher
`DETAIL_FETCHER_METRICS_PORT` (9091), `DETAIL_FETCHER_OPENDOTA_TIMEOUT_SEC` (30)
`DETAIL_FETCHER_OPENDOTA_MAX_REQ_PER_MIN` (50), `DETAIL_FETCHER_WORKER_CONCURRENCY` (5)
`DETAIL_FETCHER_WORKER_PREFETCH` (10), `DETAIL_FETCHER_WORKER_MAX_RETRIES` (3), `DETAIL_FETCHER_WORKER_RETRY_DELAY_SEC` (2)

## Parser
`PARSER_METRICS_PORT` (9093), `PARSER_WORKER_BATCH_SIZE` (100), `PARSER_WORKER_FETCH_TIMEOUT_MS` (2000)
`PARSER_WORKER_PREFETCH` (100)

## ML Trainer
`TRAINER_PATCH_ID` (0=auto), `TRAINER_MODEL_DIR` (/models)
`TRAINER_MIN_MATCHES_PER_GROUP` (5), `TRAINER_MAX_HERO_ID` (160), `TRAINER_VAL_RATIO` (0.15)
`TRAINER_AGG_BATCH_SIZE` (500), `TRAINER_PRIOR_GAMES` (3.0), `TRAINER_PRIOR_WR` (0.5)
`TRAINER_LEAGUE_ONLY` (false) — only aggregate matches with `leagueid > 0` (pro matches)
`TRAINER_LOBBY_TYPES` (empty) — comma-separated lobby_type whitelist (e.g. `7,8` for ranked)

## ML API
`API_HOST`, `API_PORT` (8080), `API_MODEL_DIR` (/models), `API_MAX_HERO_ID` (160)
`API_POOL_MIN` (1), `API_POOL_MAX` (8), `STRATZ_ADMIN_TOKEN`

## General
`LOG_LEVEL` (info), `GRAFANA_USER` (admin), `GRAFANA_PASSWORD` (admin)
